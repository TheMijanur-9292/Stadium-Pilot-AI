# StadiumPilot AI - Smart Stadium Platform

This repository contains the **Smart Stadium Platform** for FIFA World Cup 2026, including both the Next.js frontend and NestJS backend.

## Project Structure

* **`/frontend`**: Next.js client-side portal (integrated with HeroUI/TailwindCSS and Zustand).
* **`/backend`**: NestJS server-side application (integrated with Prisma ORM, Neon PostgreSQL, and Gemini AI).

## Local Development

### Backend Setup
1. Navigate to `/backend`
2. Run `npm install`
3. Configure environment variables in `.env`
4. Start development server: `npm run start:dev`

### Frontend Setup
1. Navigate to `/frontend`
2. Run `npm install`
3. Start Next.js development server: `npm run dev`
